## 1. 产品概述

养车记录工具——帮助车主轻松追踪车辆加油记录，自动计算油耗与成本，并通过趋势图直观展示油耗变化，智能标注异常波动，让养车开支一目了然。

- 目标用户：私家车主、车队管理者，需要精确追踪油耗和养车成本的人群
- 核心价值：数据驱动的油耗分析，帮助用户发现异常油耗、优化驾驶习惯、降低养车成本

## 2. 核心功能

### 2.1 用户角色
| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 普通用户 | 无需注册，本地使用 | 管理车辆、记录加油、查看分析 |

### 2.2 功能模块
1. **车辆管理页**：添加/编辑/删除车辆信息，选择当前活跃车辆
2. **加油记录页**：录入每次加油详情，自动计算百公里油耗和每公里成本
3. **油耗分析页**：油耗趋势折线图、异常波动标注、统计数据汇总

### 2.3 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 车辆管理页 | 车辆列表 | 展示所有已添加车辆，支持切换当前车辆、编辑和删除 |
| 车辆管理页 | 添加车辆表单 | 输入品牌型号、车牌号、排量、燃油类型，支持新增和编辑 |
| 加油记录页 | 记录列表 | 按时间倒序展示当前车辆的加油记录，显示油耗和成本指标 |
| 加油记录页 | 添加记录表单 | 输入加油日期、加油量、单价、总金额、当前总里程 |
| 加油记录页 | 自动计算 | 根据相邻两次里程差和加油量，自动计算百公里油耗和每公里成本 |
| 油耗分析页 | 趋势图 | 折线图展示百公里油耗变化趋势，标注异常波动点 |
| 油耗分析页 | 异常标注 | 当某次油耗高于平均值20%时，用醒目标记和提示文字标注 |
| 油耗分析页 | 统计面板 | 平均油耗、最低/最高油耗、总加油量、总里程、总花费等汇总数据 |

## 3. 核心流程

用户首次使用时添加车辆信息，之后每次加油后录入加油记录。系统根据相邻两次里程差和加油量自动计算百公里油耗与每公里成本。用户可随时查看油耗分析页面，通过趋势图了解油耗变化，系统自动标注异常波动点。

## 4. 用户界面设计

### 4.1 设计风格
- 主色调：深灰/炭黑（#1a1a2e）搭配琥珀橙（#f59e0b）强调色，营造汽车仪表盘质感
- 辅助色：冷灰蓝（#334155）用于卡片背景，翠绿（#10b981）用于正向指标，玫红（#ef4444）用于异常警示
- 按钮风格：圆角微凸（rounded-lg），主按钮使用琥珀橙填充，次要按钮半透明描边
- 字体：DM Sans 用于正文，Orbitron 用于数字/数据展示，营造科技仪表感
- 布局：卡片式布局，顶部导航栏，左侧标签切换
- 图标风格：Lucide 线性图标，2px 描边

### 4.2 页面设计概览
| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 车辆管理页 | 车辆卡片 | 深色卡片、左侧色条标识、品牌型号大字、车牌号标签、排量/燃油类型标签 |
| 车辆管理页 | 添加表单 | 模态弹窗、深色表单背景、琥珀橙提交按钮 |
| 加油记录页 | 记录列表 | 时间轴式排列、每条记录卡片显示日期/加油量/金额/里程/油耗 |
| 加油记录页 | 添加表单 | 模态弹窗、自动计算结果实时预览 |
| 油耗分析页 | 趋势图 | 深色背景折线图、琥珀橙色线条、异常点红色脉冲标记、悬浮提示 |
| 油耗分析页 | 统计面板 | 4列网格、大号数字、小号标签、图标装饰 |

### 4.3 响应式设计
- 桌面优先设计，最小宽度1024px完整体验
- 平板端（768-1024px）统计面板调整为2列
- 移动端（<768px）单列布局，底部标签导航
